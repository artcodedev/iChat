# ####################################################
# ichat_server

go to file
```bash
cd ichat_server
```

install dependencies
```bash
bun install
```

To run:
```bash
bun index.ts
```

# ####################################################
# ichat_client

go to file
```bash
cd ichat_client
```

install dependencies
```bash
bun install
```

To run dev:
```bash
bun run dev
```

To run build:
```bash
bun run build
```

To run start:
```bash
bun run start
```

# ####################################################
# docker start 

```bash
sudo docker compose up --build -d
```


This project was created using `bun init` in bun v1.3.4. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
