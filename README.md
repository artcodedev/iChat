# ####################################################
# ichat_server

Go to file
```bash
cd ichat_server
```

Install dependencies
```bash
bun install
```

To run:
```bash
bun index.ts
```

# ####################################################
# ichat_client

Go to file
```bash
cd ichat_client
```

Install dependencies
```bash
bun install
```

To run dev:
```bash
bun run dev
```

To run build:
```bash
bun run build
```

To run start:
```bash
bun run start
```

# ####################################################
# docker start 


start docker
```bash
sudo docker compose up --build -d
```
